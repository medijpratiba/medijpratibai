== Description ==
 
[medijpratiba.lv](https://medijpratiba.lv): Nereti medijpratība tiek saprasta ļoti šauri – tikai kā maldinošas informācijas atpazīšana medijos.

Šī spēle paplašina medijpratības izpratnes robežas, aptverot arī jautājumus par vārda brīvību, vizuālo pratību, naida runu, stereotipiem un attēliem medijos un citiem mūsdienās svarīgiem jautājumiem.
 
== Installation ==
 
Instalēšanas instrukcijas: [Medijpratība jautā](https://github.com/medijpratiba/medijpratiba_jauta) repozitorijā
 
Lai lietotu Medijpratiba.lv spēles WordPress spraudni, nepieciešams:

1. jau esoša WordPress mājas lapa;
2. iespēja ielādēt saturu mājas lapā. Ieteicams lietot standarta "WordPress Importer";
3. ielādēt papildus spraudni "Meta Box – WordPress Custom Fields Framework". "Advanced Custom Fields (ACF)" atbalsts var tikt pievienots nākotnē.
